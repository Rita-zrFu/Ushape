# Data Availability

The data used in this project are derived from the UK Biobank resource and are not publicly available due to data access restrictions.

This repository does **not** include any individual-level participant data.

To reproduce the real data analysis, users must:

1. Apply for access to the UK Biobank resource through the official application process  
2. Obtain the required data fields  
3. Prepare the input datasets and place them in the `data/` directory  

The code in this repository is designed to operate on user-provided UK Biobank data that follow the same structure as used in this study.
